# IMDb Analytics Dashboard — Top 1000 Movies

A cinematic, interactive data visualization project built on the IMDb Top 1000 Movies dataset.

## Overview

This project presents a comprehensive visual analysis of the IMDb Top 1000 Movies dataset,
transforming raw data into a rich, interactive storytelling experience using Tableau Public,
embedded within a custom-designed dark-themed website.

## Features

- Interactive Tableau Public dashboard (filters, drill-downs, cross-highlights)
- Cinematic dark UI with custom cursor, grain overlay, and scroll animations
- Four data-driven insight cards drawn from the dataset
- Fully responsive layout for desktop and mobile

## Tech Stack

| Tool         | Purpose                            |
|--------------|------------------------------------|
| Tableau      | Data visualization & dashboard     |
| HTML5        | Page structure & semantic markup   |
| CSS3         | Styling, animations & layout       |
| JavaScript   | Interactivity & scroll effects     |

## Dataset

- **Source:** Kaggle — IMDb Movies Dataset
- **Records:** 1,000 top-rated movies
- **Attributes:** 16 (title, year, rating, genre, director, gross revenue, etc.)
- **Year Range:** 1920–2020

## Key Insights

1. **Drama Dominates** — Over 35% of all Top 1000 entries are Drama films
2. **Stable Ratings** — Average IMDb rating holds at 8.1 across the dataset
3. **Ratings ≠ Revenue** — High-rated films rarely top the box office
4. **Longer = Better** — Films 120–180 min avg significantly higher ratings
